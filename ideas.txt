I have a PCF8575 extender with 8 buttons and 8 LEDs attached to it. Lets create a LEDController class which will allow the following:

- set the LED into ON/OFF state;
- set the LED into"slow blinking/fast blinking" state;
- set the LED into"slow blinking N times/fast blinking N times" state;
- turn all LEDs on/off



Вісім кнопок, три екрани

1  2  3  4
E1  E2  E3
5  6  7  8

Кнопки 2 та 3 перемикають режими роботи (ВІЛЬНИЙ/КРОК/НАЛАШТУВАННЯ)

На першу версію зробити два окремих режими:

ВІЛЬНИЙ
Подача енкодером чи кнопками, ніяких лімітів.

Кнопки ЗЛІВА - вперед/назад З ФІКСАЦІЄЮ
Кнопки СПРАВИ - вперед/назад БЕЗ ФІКСАЦІЇ
SHIFT:
Середній блок - налаштування швидкості подачі
Нижній блок - налаштування кроку енкодера


КРОК
Подача енкодером (?) чи кнопками покроково

Кнопки СПРАВИ - вперед/назад ПОКРОКОВО

SHIFT
Середній блок - налаштування швидкості подачі
Нижній блок - налаштування кроку на імпульс

ЛІМІТИ
